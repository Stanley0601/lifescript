# 「人生剧本」v5 产品方案 · 终稿

## 赛道：命题赛题3（QQ · AI社交）

### 赛题原文核心
> 用AI玩转QQ养虾，解锁社交"群聊出圈"、"无障碍交流"，用AI让社交更有温度

### 我们的切入点
不做养虾，做**"AI生命体验伙伴"**——一个在QQ里"活着"的AI好友，TA有自己的人生、会发动态、会主动找你聊天。你们的每一次对话，都在潜移默化中影响着彼此的选择。

### 赛道适配性论证
- QQ的核心是**社交关系**→我们做的是一种全新的AI社交关系
- 赛题鼓励"让社交更有温度"→AI好友的温度来自TA的真实感和你对TA的影响力
- 赛题鼓励"无障碍交流"→AI好友没有社交压力，是最没有障碍的倾诉对象

---

## 产品定义

**产品名**：「人生剧本」LifeScript

**一句话**：一个会在QQ里"活着"的AI好友——TA有自己的人生、会发动态、会主动找你聊天。你说的每一句话，都可能改变TA的人生。

**核心差异化（vs 普通AI聊天）**：

| 普通AI聊天 | 人生剧本 |
|-----------|---------|
| AI等你开口 | **AI主动找你** |
| AI没有自己的生活 | **AI每天在过自己的日子** |
| 对话结束就结束了 | **你的话会影响TA的人生走向** |
| AI是工具 | **AI是一个你会牵挂的人** |

## 功能架构

### 🏠 主界面：QQ消息列表
仿QQ消息列表页，展示所有AI好友的对话入口。每个好友显示：
- 头像 + 名字
- 最后一条消息预览
- 时间戳
- 未读消息数
- 在线状态（"在图书馆"/"在逛街"/"睡觉了"）

### 💬 聊天界面：像素级QQ还原
- 顶部：好友名字 + 在线状态
- 消息区：左右气泡 + 头像 + 时间分隔线
- 角色消息一条条蹦出，有"正在输入..."
- 底部：输入框 + 表情按钮(装饰) + 发送按钮
- 建议回复条（输入框上方，3个可点击）
- 语音消息条（角色偶尔发语音，显示为绿色条状）

### 📱 朋友圈/空间动态
仿QQ空间动态流：
- 角色会发动态（文字+图片描述）
- 显示发布时间、点赞数
- 你可以点赞和评论
- 动态内容和角色当前的人生阶段相关
  - 如：纠结期发"失眠了 在想一些事情"
  - 做出决定后发"新的开始✨"
  - 一年后发"公司拿到A轮了！！"

### 👤 角色资料页
仿QQ个人资料页：
- 头像 + 背景图
- 个性签名（随剧情变化）
- 基本信息（年龄/学校/专业）
- "TA的人生故事"按钮 → 查看已解锁的剧情时间线

### 🔔 角色主动消息
角色不只是等你说话，TA会主动发消息：
- 进入app时可能已经有未读消息
- 消息内容和TA当前的生活阶段相关
- 如果你很久没回复，TA会再来问问

## 体验流程

```
打开App
  │
  ▼
消息列表（QQ风格）—— 看到3个AI好友
  │
  ▼
点击某个好友 → 进入聊天界面
  │
  ├── 如果是第一次：看到好友申请系统消息 + TA的自我介绍
  │
  ├── 如果不是第一次：看到TA的主动消息（"在吗！跟你说个事"）
  │
  ▼
自由聊天（打字 或 点建议回复）
  │
  ▼
AI根据你的话推进剧情阶段（你不知道的）
  │
  ▼
聊到关键节点 → TA做出人生决定 → 时间跳跃 → 看到后续
  │
  ▼
返回消息列表 → 朋友圈出现新动态 → 角色状态更新
  │
  ▼
结局出现 → 「TA的人生故事」解锁完整时间线
```

## UI规范

### 配色（QQ风格）
- 主背景：#f5f5f5
- 聊天背景：#f0efe5（和微信一致，QQ新版也趋近）
- 自己的气泡：#95ec69（绿色）
- 对方的气泡：#ffffff
- 强调色/按钮：#12b7f5（QQ蓝）
- 顶部导航栏：#ededed
- 字体：PingFang SC / -apple-system
- 头像：40×40px 方形圆角6px

### 字号（严格对标）
- 聊天消息正文：16px
- 好友名字（列表）：16px semibold
- 消息预览（列表）：14px #999
- 时间戳：12px #bbb
- 系统消息：12px #999
- 旁白/提示：13px #aaa italic

### 间距
- 消息之间：8px
- 同一人连续消息：4px
- 气泡内padding：10px 14px
- 头像与气泡间距：8px
