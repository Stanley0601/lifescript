一、产品概述

解决什么问题？

当代年轻人面对人生重大选择（升学vs就业、创业vs稳定、兴趣vs现实）时，缺乏一个可以坦诚交流、没有社交压力的对象。现有AI助手要么过于工具化，要么缺乏情感深度。

我们的方案

「人生剧本」是一个在QQ里"活着"的AI好友。TA有自己正在经历的人生困境——而你作为一个刚被加上的陌生人，你说的每一句话都在潜移默化中影响着TA的选择，最终改变TA的人生走向。

核心差异化

千人一面基于你的兴趣个性化匹配话题

赛道适配（QQ社交 · 赛题3）

  • QQ的核心是社交关系 → 我们做的是一种全新的AI社交关系
  • 赛题鼓励"让社交更有温度" → AI好友的温度来自TA的真实感和你对TA的影响力
  • 赛题鼓励"无障碍交流" → AI好友没有社交压力，是最没有障碍的倾诉对象

二、核心创新点

1. 兴趣感知的主动社交

用户进入时选择兴趣标签（足球/动漫/音乐/游戏等10类），系统从40+热点话题池中匹配最合适的破冰内容。AI好友会主动发来和你兴趣相关的消息，而不是等你开口。

2. 关系递进系统

对话过程中跟踪亲密度(familiarity)和默契度(chemistry)，关系会从"陌生→熟络→暧昧"递进。不同关系阶段下，角色的说话方式、动态内容、推荐理由都会变化。

3. 活人感设计（参考腾讯AIMate理论）

  • 日常生活系统：角色会发"去图书馆了""刚跑完步"等日常状态
  • 天气感知：对话中自然融入天气话题（"今天好热啊"）
  • 朋友圈动态：角色发动态和你兴趣强相关，你可以点赞评论
  • 情绪延续：角色的情绪会从上次对话延续到下次

4. 旁观者叙事设计

用户不是故事的主角，而是旁观者/倾听者。不需要做选择题，只需要自然聊天。你的态度（鼓励/中立/劝退）会被系统识别，最终影响角色的人生走向，产生9种不同结局。

5. 结局后的启发反思

故事结束后展示完整的人生时间线、数值化的人生指标（幸福/成长/稳定/勇气/连接），以及"如果你说了不同的话"的平行结局对比，引发用户对自己人生选择的反思。

核心算法

  • 意图分类：关键词加权匹配（encourage/question/discourage/neutral）
  • 阶段推进：轮次计数 + 意图触发双重条件
  • 结局决定：全程意图历史加权投票
  • 兴趣匹配：用户标签 × 角色性格 → 话题池筛选 + 亲和度评分
  • 关系递进：基于对话轮次和互动行为的亲密度/默契度增长模型

推荐体验路径（3-5分钟）

  1. 打开 Demo → 点击「进入QQ」
  2. 选择一个角色的人生（推荐林小宇，保研vs创业更贴近大学生）
  3. 选择你的兴趣标签（选2-3个）
  4. 进入消息列表 → 看到AI好友的主动消息 → 点击进入聊天
  5. 自由聊天 或 点击建议回复
  6. 体验完整结局 → 查看人生时间线
  7. 切换到「动态」Tab → 看角色的朋友圈
