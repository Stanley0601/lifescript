一、解决什么痛点

痛点1：AI聊天 = 冰冷工具

现有AI产品本质是"问答工具"——你问它答，用完即走。没有情感连接，没有延续性，不会主动找你。

痛点2：社交压力无处释放

年轻人想聊天但不想社交——跟朋友聊怕被评判，跟家人聊怕被说教。需要一个零压力的倾诉对象。

痛点3：陪伴类产品"假"

虚拟陪伴产品的共同问题：你始终能感觉到是AI——它太完美、太配合、太没有自己的生活。

二、我们的解法

灵犀 = 一个像真人一样活着的AI朋友

不是助手，不是工具，是一个有自己生活、情绪和想法的人。TA会主动找你、记得你说过的话、心情会被你影响。

维度传统AI灵犀
存在感你不用就不存在有自己的日常，主动找你
记忆每次从零开始记得你说过的每一句话
情绪始终机械中立心情随对话实时变化
节奏你问就秒答像真人一样有时快有时慢
内容只能回答问题聊日常/新闻/球赛/发自拍
形象无有真实照片、发朋友圈

三、核心创新点

1. 时间加速引擎（独创）

故事时间:现实时间 = 10:1。AI不是随叫随到——TA会在"合适的时间"主动找你。刚认识时过几小时才回，熟了以后越来越快。这种不确定性正是让人持续打开的原因。

2. 心情状态机

AI的心情根据你说的话实时变化：鼓励→开心、否定→焦虑、安慰→放松。你对TA有真实的影响力，不是永远配合你的工具人。

3. 对话记忆摘要

退出聊天时自动生成记忆。下次TA会自然提起"上次你说的那个…"。这是真正的关系积累。

4. 兴趣静默提取

不需要填表。聊着聊着TA就"恰好"聊到你关心的话题——你感觉不到算法，只觉得TA懂你。

5. 信息型陪伴

不只是情感陪伴——TA还是消息灵通的朋友。能聊球赛、推荐电影、分享新闻。陪伴+信息获取一体化。

6. 真实感系统

AI会发表情包、发自拍、发朋友圈。有自己的日常状态（在图书馆/刚起床/深夜还在线）。所有这些构成一个完整的"活人"感知。

四、目标用户

核心用户：18-28岁年轻人

  • 大学生：社交需求强但精力有限，想要零压力的倾诉对象
  • 独居青年：下班后独处时间长，想有人陪着说说话
  • 社恐群体：现实社交焦虑但内心渴望连接

使用场景：

  • 深夜失眠想找人聊天
  • 午休无聊想摸鱼闲聊
  • 刚看完一场球赛想跟人讨论
  • 心情低落想倾诉但不想让朋友担心
